*This is a Discord Music bot made by anandvsinh.* 

*This bot uses youtube-api to stream music in Discord VC's in realtime.*

Below is the installation, configuration process:

### Installation:
```
git clone https://github.com/anandvsinh/discord-music-bot
cd project-folder
npm install
```

### Requirements / Dependencies:
```
Node.js 20.x.x (or newer)
FFMpeg 5.3.0 (or newer)
Shoukaku 3.4.2 (or newer)
lavalink-client 2.9.1
```

### Working:
Replace 
```
Discord_bot_token = "Enter your own token"
```
